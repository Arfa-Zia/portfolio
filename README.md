# Arfa Zia Portfolio

A modern, clean personal portfolio website for **Arfa Zia** (Mobile App Developer), built with **Next.js App Router**, **Tailwind CSS**, and **Framer Motion**.

## Live Purpose

- Showcase who Arfa is and the type of mobile apps she builds
- Highlight featured projects and technical strengths
- Encourage visitors to start a project through a strong call-to-action

## Tech Stack

- Next.js (App Router, JavaScript)
- Tailwind CSS
- Framer Motion

## Pages

- `/` - Home
- `/about` - About
- `/projects` - Projects

## Reusable Components

- `components/Navbar.js`
- `components/Hero.js`
- `components/AboutSection.js`
- `components/ProjectsSection.js`
- `components/ProjectCard.js`
- `components/CTA.js`
- `components/ContactForm.js`
- `components/Footer.js`

## Run Locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Production Build

```bash
npm run build
npm run start
```

## Deployment

This project is ready to deploy on [Vercel](https://vercel.com).
